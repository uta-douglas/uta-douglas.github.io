# MyMav Documentation and Testing

This directory (`mymav/`) serves as a transient location for hosting documentation and conducting testing related to the MyMav system.

## Purpose

- **Documentation Hosting:** Contains HTML files and associated assets for temporary web viewing of MyMav documentation, such as step-by-step guides (e.g., `EnrollDrop.html`).
- **Testing Environment:** Used for various testing purposes, including load testing or functional testing of MyMav processes. The `2026-load-test/` subdirectory, for example, holds resources for specific testing scenarios.

## Contents

- `2026-load-test/`: Directory containing resources for load testing, including HTML pages and supporting images.
- Other HTML files (e.g., `EnrollDrop.html`): Direct access to documentation pages.
- `README.md` (this file): Provides an overview of the directory's purpose and contents.

**Note:** The content within this directory is intended for temporary use and may be updated or removed frequently.
