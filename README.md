# uta-douglas.github.io

This repository serves as a collection of web-related projects and tests hosted on GitHub Pages.

## Projects

Below is a list of the projects currently available in this repository:

- **Druid:** Contains web snippets and test pages related to the Druid data analytics platform.
- **MyMav:** Includes materials for load testing the MyMav system, with detailed steps and images from the enrollment and drop processes.
- **Threatfinder Test:** A test suite for Threatfinder, featuring sample XML data and XSLT transformations.
- **XSLT:** A collection of sample XSLT files for various XML transformations.

Each project folder contains its own documentation and resources. For more information, please refer to the `README.md` file within each respective directory.
